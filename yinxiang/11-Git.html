        历史
                查看历史
                        git reflog
                                在Git中，用HEAD表示当前版本
                                上一个版本就是HEAD^
                                再上一个版本就是HEAD^^
                                往上100个版本可以简写为    HEAD~100
                比较不同
                        git diff *********
                                git diff HEAD -- df_user/models.py
                                        对比工作区和仓库区某版本    某文件或某目录下      的不同
                                git diff HEAD HEAD^^    -- df_user/models.py
                                        对比上次提交和上上次提交
                                git diff SHA1 SHA2
                                        比较两个历史版本的区别
                回退
                        git reset HEAD^或版本号
                                将仓库区中某个版本的修改恢复到暂存区
                        git status
                        git checkout -- df_user/models.py
                                将暂存区的恢复到工作区
