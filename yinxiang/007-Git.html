• git submodel
    • 将另一个仓库作为子模块
    • https://www.jianshu.com/p/9000cd49822c
    • git submodule update --remote
      • 更新子模块到远程仓库最新的
  • git cherry-pick xxxxx
    • 拉取A分支中的一个提交版本到B分支
    • 提交时按功能提交，不要一次修改很多代码再修改
  • 撤销
    • git reset
      • 在私有分支上舍弃一些没有提交的更改
      • 将文件从缓存区中移除
    • git revert
      • 在共有分支上回滚更改
    • git checkout
      • 切换分支或查看就版本
      • 舍弃工作目录中的更改
  • 重新历史

      • 重写历史
    • git push  —-force origin
      • 覆盖远程版本历史
