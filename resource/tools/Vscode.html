# settings

  • 设置文件隐藏
  • files.exclude
  • 标尺
  • editor.rulers
  • 一个制表符等于的空格数
  • editor.tabSize
{
    "python.pythonPath": "xx",
    "editor.rulers": [80],
    "editor.tabSize": 4,
    "files.exclude": {
        "**/__pycache__": true  
    },
    "python.linting.flake8Args": ["--max-line-length=120"]
}
